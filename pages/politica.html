Política de Privacidad

Transparencia en el uso de la información y respeto por la ciudadanía.

Este sitio web está orientado a facilitar el acceso a información y recursos públicos. No recopila información personal de los usuarios de forma automática.

Recolección de información

Este sitio no almacena datos personales directamente. La interacción con enlaces externos (como WhatsApp u otras plataformas) depende de las políticas de dichos servicios.

Uso de enlaces externos

Algunos enlaces dirigen a plataformas de terceros. Este sitio no tiene control sobre el manejo de datos realizado por dichas plataformas.

Seguridad

Se implementan buenas prácticas de desarrollo para garantizar la integridad del sitio, aunque no se gestionan bases de datos de usuarios dentro de esta plataforma.

Actualizaciones

Esta política puede ser modificada en cualquier momento para adaptarse a cambios en el funcionamiento del sitio o nuevas funcionalidades.

El objetivo de este sitio es facilitar el acceso a información confiable, sin recolectar datos personales innecesarios.
