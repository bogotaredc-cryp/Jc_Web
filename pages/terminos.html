Condiciones para el uso responsable del sitio web.

Al acceder a este sitio web, aceptas las condiciones establecidas en este documento. Este espacio está orientado a la participación ciudadana, el acceso a información y la construcción colectiva.

El contenido del sitio es de carácter informativo y participativo. El usuario se compromete a hacer un uso adecuado y respetuoso.

Los textos, imágenes y materiales del sitio tienen fines informativos. No deben ser utilizados de forma indebida o con fines engañosos.

El sitio puede contener enlaces a plataformas externas. No se tiene control sobre sus contenidos ni políticas.

Conducta del usuario

No está permitido el uso del sitio para actividades ilegales, ofensivas o que afecten a terceros.

Modificaciones

Estos términos pueden actualizarse en cualquier momento sin previo aviso.

El uso de este sitio implica la aceptación de estos términos. Nuestro objetivo es promover un espacio respetuoso, abierto y transparente.
